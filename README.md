# React FireBlock App

Responsive FrieBlock App made using React and Bootsrap.

<hr />

## What I Used

- Firebase
- React Toastify
- React-Bootstrap



### Installation Options

```
$ git clone https://github.com/farukkursun/firecontact
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/farukkursun/firecontact
```

```
$ yarn
```

```
$ yarn start
```
